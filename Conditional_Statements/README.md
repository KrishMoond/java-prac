### Conditional Statements
```java
// If-else statement
if (condition) {
    // True block
} else {
    // False block
}

// Switch statement
switch (variable) {
    case value1:
        // Code block
        break;
    case value2:
        // Code block
        break;
    default:
        // Default block
}
```

### Type Casting
```java
// Widening (Implicit)
int num = 10;
double d = num;  // Automatic conversion

// Narrowing (Explicit)
double d = 10.5;
int num = (int) d;  // Manual conversion
```
